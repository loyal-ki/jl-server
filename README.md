### Installation

1. Clone source code
```bash
$ git clone https://github.com/loyal-ki/jl-server.git
$ cd journey-lingua-backend
```

2. Initial setup && build the Docker image
```bash
$ make initialize
```

3. Run source
```bash
$ make up
```

4. Install DBeaver && Connect to MySQL
- Server host: localhost
- DB name: journey-lingua
- user: local
- password: local
- port 3306
